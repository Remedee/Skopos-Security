From left to right:

Mia Ward, Zachary Edwards Downs, Merrell Reed, Todd Robinson, and Sinan Fathulla

We are a group of Computer Science students from Texas Tech University. For our Capstone Project, we have challenged ourselves to make a security system to help deter package thieves and burglars alike. Using Apache, MySQL, HTML, PHP, CSS, Python’s OpenCV library, and ✧a little bit of magic✧, we are able to make a fully functional website client paired with a Raspberry Pi 3+ sporting a motion detection/image recognition software. If you have any questions, feel free to visit our Github for more information on the backend of our nifty project!
